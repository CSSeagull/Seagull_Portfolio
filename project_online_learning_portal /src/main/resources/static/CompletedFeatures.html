Key Website Features

  • Quiz Logic: One of the key features of our website is quiz logic: Teachers can create quizzes and dynamically add any number of questions with the correct answer option. The initially created quiz will be hidden from students. Therefore, the teacher will have the opportunity to open it or hide it again in their panel. As soon as the professor reveals the quiz, it will appear in the student's dashboard, where the student can go to the quiz and solve it. If the student clicks the "Submit quiz" button, their result will be saved in the database and the quiz will no longer be available for the student to take again.
  • Material Upload Logic: Another key feature that was implemented is material upload logic: Teachers can upload files to the course that are visible to students. This is done to make the learning process easier for students, as all the materials they need are concentrated in one place.
  • User Authentication and Authorization: User authentication and authorization: added login and registration pages, therefore students can only view courses they are enrolled in.
  • Course Catalogue and Enrollment: Course catalogue and enrollment: Students can choose any course to enroll in from the catalogue.
  • Progress Tracking: Progress tracking: student can see his/her progress towards the course by comparing his/her grades in different quizzes.
